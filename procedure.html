Procedures

Using a numbered list, describe your proecedure w/ pictures


Step

Description

Image

1

Preprocessing: MNIST images were grayscaled, normalized, and resized to make them suitable for kernel operations

2

Set kernel size (3x3, 5x5, or 7x7)

3

Feature extraction: convolution (64 filters) + ReLU → 2x2 max pooling → convolution (64 filters) + ReLU → 2x2 max pooling → flattening

4

Feature classification: fully connected layer (64 neurons) w/ dropout (p = 0.5) + ReLU → fully connected layer (32 neurons) w/ dropout (p = 0.5) + ReLU → output layer (10 neurons) + softmax activation

5

Repeat steps 2-4 for all kernel sizes.

6

Training
Loss function: sparse categorical cross entropy
Optimizer: Adam
K-fold cross-validation using 10 folds
Metrics (accuracy and loss) were collected during training

7

Data analysis: Microsoft Excel was used to calculate mean, standard deviation, and variance for the accuracy and loss data. A single factor ANOVA test was used to determine statistical significant difference. If a significant difference was found, Tukey's Range Test was performed as a post hoc analysis to determine the significant difference between the groups.
